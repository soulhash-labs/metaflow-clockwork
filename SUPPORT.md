# Support

## Usage Questions

Use GitHub Issues for usage questions, bug reports, and feature requests unless GitHub Discussions is enabled later.

## Bug Reports

Use the bug-report issue template and include:

- Python version
- install method
- command run
- expected behavior
- actual behavior
- relevant logs or traceback

## Security Issues

Do not use public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Scope

This repository currently provides community-style support only. There is no public SLA.
